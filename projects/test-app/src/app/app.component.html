Standard

Intially Expanded Panel

Using [id] and [activeIds] sets the panel to be expanded.

Use (panelChange) to catch when the panel is toggled.

 Disabled Panel

Setting [disabled] to true stops the user from being able to toggle the panel.

 Final Panel

This is the final panel.

Opening this one doesn't affect the others as [closeOthers] is set to false.
